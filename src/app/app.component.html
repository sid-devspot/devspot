Project

Walrec

WalRec is a personal finance tracking system designed to simplify how users record, understand, and manage their money — across both web and mobile platforms.

Managing money shouldn't feel complicated. 🔗 Live App

The Idea

The goal behind WalRec was to create a system that feels simple, guided, and easy to use — while still being powerful enough to manage real financial scenarios.

Core Features

Dashboard & Insights

Provides a quick overview of income, expenses, and balance along with visual charts like cashflow, category analytics, and growth tracking.

Transaction Management

Users can add, edit, and filter transactions with ease. The integration of calendar-based filtering helps users track their finances by date.

Budget Planner (EMI & Subscription)

A dedicated planner to track EMIs and subscriptions with due dates, progress tracking, and payment history.

Category Customization

Users can create and manage their own categories with custom icons and colors, making the experience more personal and organized.

Profile Personalization

Includes avatar selection and profile customization to make the app feel more user-centric.

TravellersPond

TravellersPond is a travel booking platform designed to help users explore, compare, and book travel packages through a clean and intuitive web experience.

Exploring travel should feel exciting, not complicated. 🔗 Live

The Goal

Simplify travel discovery, provide structured information, and enable a smooth booking journey across all devices.

Core Features

A complete platform for browsing, understanding, and booking travel experiences.

Tour Listings

Browse a variety of travel packages with visually rich and structured listings.

Detailed Package View

Each tour includes itinerary, pricing, and highlights for better decision making.

Inquiry & Contact

Quick inquiry system to connect users with the travel agency easily.

Responsive Experience

Optimized layouts for both desktop and mobile devices.

In addition to user-facing features, the platform includes a custom admin system for managing content dynamically.

Advanced Feature

Dynamic Admin & Live Preview System

A custom-built admin panel that enables real-time content editing with live preview capabilities.

To make the platform scalable and easy to manage, I developed a custom admin application that enables dynamic control over website content without requiring code changes. The system offers a smooth editing experience with instant preview capabilities, allowing users to see updates in real time while modifying content. It includes flexible dynamic forms for managing tours, banners, and pages, a live preview system using iframe, instant synchronization of changes to the live website, and reusable content blocks designed for scalability and future expansion.

Next Project

CssFusion

cssfusion is a lightweight CSS utility package designed to simplify styling, speed up development, and maintain consistent UI across projects.

Build faster. Style smarter. 🔗 NPM Package

The Idea

The goal behind cssfusion was to reduce repetitive CSS work by creating a set of reusable utility classes and styling patterns, allowing developers to build interfaces quickly and consistently without writing styles from scratch for every component.

Core Features

Utility-Based Styling

Predefined classes for layout, spacing, and typography to speed up UI development.

Reusable Patterns

Common UI structures that help maintain consistency across different screens.

Responsive Helpers

Built-in support for responsive design to adapt layouts across devices.

Outcome

cssfusion improved development speed and made it easier to build scalable, consistent user interfaces across multiple projects.
